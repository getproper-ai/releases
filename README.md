## Proper Releases

**This repository contains release artifacts and installers only.**

- Website: https://getproper.ai
- Source code is private and is **never** published here or in any public releases.

All files attached to releases in this repo are platform-specific binaries (`.dmg`, AppImage, `.exe`, `.msi`, etc.) plus their SHA-256 checksums.

GitHub may automatically display "Source code (zip)" / "Source code (tar.gz)" links on release pages. These archives contain only the minimal contents of *this* repository (the README and any artifacts we explicitly control). They do **not** contain Proper source code.

Download only the explicitly named platform installers and checksum files.
